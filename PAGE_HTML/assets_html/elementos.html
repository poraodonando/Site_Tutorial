Elementos em HTML

Tag < a >

O elemento <a > em HTML (ou elemento âncora), com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada <a > precisará indicar o destino do link.

Atributos

  • download - Leva o usuário a salvar a URL em vez de navegar até ela. Pode ser usado com ou sem um valor
  • href - A URL para a qual o hiperlink aponta. Links não se restrigem a URLs baseadas no protocolo HTTP — eles podem utilizar qualquer tipo de URL suportado pelos browsers
  • target - aonde o link vai abrir
    • _self - Na propria pagina
    • _blank - abre uma nova aba
